Public JSON API

Build on the fight predictor.

Use the API to list fighters, inspect profile features, retrieve thumbnails, request matchup predictions, and expand defeat trees.

List fighters

GET /api/v1/fighters?limit=25&search=cruz

Returns prediction-ready and provisional fighter profiles with model input fields and thumbnail URLs.

Get one fighter

GET /api/v1/fighters/1

Returns one fighter profile by ID.

Defeat tree

GET /api/v1/fighters/1/defeat-tree?depth=4

Returns nested winner-to-defeated-opponent branches after fight-result edges are imported.

Generated thumbnail

GET /api/v1/fighter-thumbnail.svg?name=cruz

Returns a deterministic fallback thumbnail for any fighter name.

Predict matchup

POST /api/v1/predict 
{
  "fighter_a_id": 1,
  "fighter_b_id": 2
}

Example response

{
  "prediction": {
    "winner": "Alex Mercer",
    "probability_a": 0.62,
    "probability_b": 0.38,
    "confidence": 0.62
  }
}
