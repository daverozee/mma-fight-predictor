Public JSON API

Build on the fight predictor.

Use the API to list fighters, inspect profile features, and request matchup predictions from two fighter IDs.

List fighters

GET /api/v1/fighters?limit=25&search=cruz

Returns prediction-ready and provisional fighter profiles with model input fields.

Get one fighter

GET /api/v1/fighters/1

Returns one fighter profile by ID.

Predict matchup

POST /api/v1/predict 
{
  "fighter_a_id": 1,
  "fighter_b_id": 2
}

Example response

{
  "prediction": {
    "winner": "Alex Mercer",
    "probability_a": 0.62,
    "probability_b": 0.38,
    "confidence": 0.62
  }
}
